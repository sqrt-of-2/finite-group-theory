# Finite Group Explorer

**Finite Group Explorer** is a tool to explore the properties of finite groups. It visualizes group structures and computes properties.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sqrt-of-2/finite-group-theory.git
   cd finite-group-theory
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:5173`.

## Testing

The project includes a test suite using **Vitest**.

To run the tests:
```bash
npm run test
```

## Technologies

- **Frontend**: React, TypeScript, Vite
- **Math Rendering**: KaTeX
- **Testing**: Vitest, React Testing Library
- **Styling**: Plain CSS
